# Valusis Project

This is a business valuation assistant.

## How to Run

1.  Make sure you have Python 3 installed.
2.  Open your terminal in the project's root directory.
3.  Run the following command to start a simple web server:

    ```bash
    python3 -m http.server 8000
    ```

4.  Open your web browser and go to `http://localhost:8000` to see the project running.
